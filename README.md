# Fundamentals of Java & Web Programming – Learning Repository

This repository contains the programs, exercises, and practical implementations developed during the training program:

Course: Fundamentals of Java & Web Programming
Organization: Anudip Foundations
Duration: September 2024 – February 2025

The purpose of this repository is to document the practical learning and hands-on implementation of concepts covered throughout the internship training, including core programming, database connectivity, ORM frameworks, and testing practices.

## Course Overview

The Fundamentals of Java & Web Programming course focused on building a strong foundation in backend and web development using Java and related technologies. The training combined theoretical understanding with practical implementation of real-world development concepts.

Throughout the program, multiple exercises and example implementations were developed to strengthen programming logic, software design understanding, and development best practices.

## Topics Covered

### Core Java

* Java syntax and program structure
* Variables and data types
* Operators and expressions
* Conditional statements
* Looping constructs
* Methods and functions
* Arrays and string handling
* Exception handling
* File handling
* Java Collections Framework

### Object-Oriented Programming

* Classes and objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Method overriding and method overloading

### Database Connectivity (JDBC)

* JDBC architecture
* Connecting Java applications to relational databases
* Executing SQL queries
* Performing CRUD operations
* Prepared statements and parameterized queries

### Hibernate (ORM Framework)

* Hibernate architecture
* Configuration and setup
* Entity mapping
* Hibernate annotations
* Session and transaction management
* Performing CRUD operations using Hibernate

### Unit Testing

* Introduction to unit testing
* Writing test cases using JUnit
* Understanding testing principles and best practices

### Mockito

* Mocking dependencies in unit tests
* Writing unit tests with Mockito
* Verifying method interactions

## Skills Gained

* Strong understanding of core Java programming
* Application of object-oriented programming principles
* Database connectivity using JDBC
* ORM implementation using Hibernate
* Writing unit tests using JUnit
* Mocking and testing with Mockito
* Understanding of backend development workflows
* Code organization and development best practices

## Learning Outcomes

By completing this training program, the following competencies were developed:

* Ability to write structured and maintainable Java programs
* Understanding of database integration with Java applications
* Practical knowledge of ORM frameworks for data persistence
* Experience with unit testing and testing frameworks
* Improved problem-solving and logical thinking skills
* Familiarity with the software development lifecycle and debugging practices
